## Completed

### 1. Signal Infrastructure (Priority 1)

Full POSIX signal syscall support added to AIOS:

- **include/aios/aios_signal.h**: Signal types, constants (22 signals),
  per-process signal state (aios_sigstate_t), kernel-format sigaction
- **aios_posix.c**: 5 new syscall handlers:
  - rt_sigaction: install/retrieve signal handlers, rejects SIGKILL/SIGSTOP
  - rt_sigprocmask: SIG_BLOCK/SIG_UNBLOCK/SIG_SETMASK with SIGKILL/SIGSTOP protection
  - rt_sigpending: query pending signal bitmask
  - kill: route to pipe_server via PIPE_SIGNAL IPC (label 75)
  - tgkill: single-threaded variant, delegates to kill path
- **pipe_server.c**: PIPE_SIGNAL handler:
  - sig 0: POSIX existence check (kill -0 pid)
  - SIGKILL(9)/SIGTERM(15): immediate process_kill()
  - Others: set sig_pending bitmask on target active_proc
- **root_shared.h**: PIPE_SIGNAL (75), sig_pending field in active_proc_t
- **signal_test.c**: 14/14 tests pass (sigaction, sigprocmask, kill, SIG_IGN)

### 2. Modularize aios_root.c (Priority 2)

Root task refactored from 674 lines to 187-line orchestrator:

- **src/boot/blk_io.c** (89 lines): blk_read_sector, blk_write_sector

- **src/boot/spawn_util.c** (55 lines): spawn_with_args, spawn_simple

  (boot_fs_init), Phase 3 (boot_start_services), main UART polling loop
