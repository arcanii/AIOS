## Completed in v0.4.60

* Signal Phase 4 -- EINTR on interruptible syscalls: aios_sig_check()
  now returns int (count of dispatched signals). read(), write() return
  -EINTR when a user-installed handler fires on entry. nanosleep()
  returns -EINTR with remaining time in rem. Ignored signals (SIGCHLD
  default-ignore, SIG_IGN) do not increment dispatched count, avoiding
  false EINTR on every post-fork write. posix_verify 79/79 confirmed.
  Files: src/lib/aios_posix.c, src/lib/posix_internal.h,
         src/lib/posix_file.c, src/lib/posix_time.c

## Priority 1: sbase rm stat error

sbase rm calls stat() before unlink. Our fstatat returns -ENOENT
for files that exist on disk, causing a spurious error message
even though unlink succeeds. Likely fetch_stat IPC path issue
for files created in the current session.
Files: src/lib/posix_stat.c, src/aios_root.c (fs_thread stat)

## Priority 2: PSCI Shutdown

Add PIPE_SHUTDOWN (label 77) IPC handler in aios_root.c that
issues PSCI SYSTEM_OFF (SMC 0x84000008) to cleanly power off
the QEMU virt machine. New aios_shutdown program sends the IPC.
Files: src/aios_root.c, src/apps/aios_shutdown.c

## Priority 3: Shared-memory pipes

Replace IPC round-trip with vka_alloc_frame mapped into both VSpaces.
Files: src/servers/pipe_server.c

## Priority 4: Allocator Memory Audit

The bump from 800 to 1500 pages is a band-aid. Audit untyped
consumption across boot, exec, fork, and thread creation to
right-size the pool.
